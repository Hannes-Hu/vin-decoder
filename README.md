# VIN Decoder Website

A modern VIN decoder website built with Vue.js, Tailwind CSS, and Node.js, using the NHTSA API.

## Features
- Real-time VIN validation
- Responsive design
- Vehicle specification details
- Safety information
- Netlify deployment ready

## Tech Stack
- Frontend: Vue.js + Tailwind CSS
- Backend: Node.js + Express
- API: NHTSA VIN Decoder

## Deployment

This project is configured for easy deployment on Netlify.
